# Shady Chat 💬  
Shady Chat is a real-time communication platform built using the MERN stack (MongoDB, Express.js, React, Node.js) and Socket.IO. This project was undertaken to explore and learn about implementing **real-time communication**, **backend configurations**, and **Socket.IO** for efficient client-server interaction.

## 🚀 Features  
- **Real-Time Messaging**: Seamless real-time communication between users.  
- **Notifications**: Get real-time notifications for messages and activities.  
- **Group Chat**: Create and participate in group conversations.  
- **User Search**: Search for other users to initiate private chats.  
- **Responsive UI**: Designed using Chakra UI for a sleek and user-friendly interface.  

## 🔧 Technology Stack  
- **Frontend**: React.js, Chakra UI  
- **Backend**: Node.js, Express.js, Socket.IO  
- **Database**: MongoDB  
- **Other Tools**: Axios for API requests, Local Storage for user session management

## ⚡ Setup and Run  
1. Clone the repository:  
   ```bash
   git clone https://github.com/yourusername/shady-chat.git
   cd shady-chat
   ```
2. Install dependencies:  
   ```bash
   npm install
   cd client
   npm install
   ```
3. Configure your `.env` file for backend variables like MongoDB URI and JWT secret.  
4. Start the development server:  
   - Backend: `npm start`  
   - Frontend: `npm start` (inside the `client` directory)
